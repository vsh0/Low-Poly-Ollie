# Low-Poly-Ollie
<img width="1920" height="1080" alt="lowpollie spread" src="https://github.com/user-attachments/assets/f9b0c4f1-cb43-4df3-ab80-d4e242f2aa32" />

For this project I decided to create a low poly model of my character Ollie. I chose this style because I enjoy the look of 3D imitating 2D and wanted to learn how to emulate that style. I also wanted to make a model that would be light-weight and easy to render. I would say overall it was a success and I'm happy with how it came out! The modelling went smoothly and rendering took no time at all (I almost shed a tear). Rigging was a challenge but I did eventually figure it out.

This was my first time rigging and animating a character model!
